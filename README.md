# 📬 업무 메일 자동 분류 시스템

> 텍스트 마이닝 기반 카테고리 분류 및 PDF 관리  
> 김민석 · 윤여옥

---

## 📌 프로젝트 개요

업무 메일이 폭증하면서 발생하는 세 가지 핵심 문제를 해결하기 위해 기획된 시스템입니다.

| 문제 | 설명 |
|------|------|
| 메일 폭증 | 중요한 메일을 놓치는 경우 발생 |
| PDF 관리 어려움 | 어느 메일에 어떤 첨부파일이 있는지 찾기 어려움 |
| 회신 누락 | 회신이 필요한 메일을 처리하지 못하는 경우 발생 |

**목적**
- 업무 메일 증가로 인한 중요 메일 누락 및 회신 지연 문제 해결
- PDF 첨부파일을 카테고리별로 자동 정리하여 문서 관리 효율화

## ⚙️ 시스템 전체 흐름

```
1. IMAP 접속      →  Gmail 서버 연결
2. 메일 수집      →  제목 · 발신자 · 본문 · 첨부파일
3. 텍스트 추출    →  제목 · 본문 · 첨부파일(PDF)
4. 형태소 분석    →  KoNLPy 명사 추출
5. 분류           →  FreqDist / TF-IDF
6. 저장           →  첨부파일 · CSV · WordCloud 이미지
```

---

## 🔍 구현 상세

### 1. 데이터 수집 — IMAP / pdfplumber
- IMAP으로 Gmail 계정에 접속하여 메일 수집
- pdfplumber를 사용해 PDF 첨부파일에서 텍스트 추출
- 제목, 본문, PDF 텍스트를 통합하여 분석에 활용

### 2. 전처리 — KoNLPy / 불용어
- KoNLPy를 이용한 형태소 분석
- 명사 추출 후 불용어 제거

### 3. 수치화 — FreqDist / TF-IDF
- 단어 빈도(FreqDist) 및 TF-IDF 가중치 계산

### 4. 분류 — 카테고리 분류
메일을 아래 5개 카테고리로 자동 분류합니다.

| 카테고리 | 설명 |
|----------|------|
| 업무협조 | 협조 요청 관련 메일 |
| 보고서 | 결과 보고 및 분석 자료 |
| 회의록 | 회의 관련 메일 및 첨부파일 |
| 공지 | 사내 공지 및 안내사항 |
| 기타 | 위 카테고리에 해당하지 않는 메일 |

### 5. 시각화 — WordCloud
- 메일 본문과 PDF 텍스트에서 핵심 키워드 추출
- 불용어 제거 후 워드클라우드로 시각화

## 📊 주요 기능

### 메일 분류 결과 CSV 저장
- 제목, 발신자 등 메일 기본 정보 저장
- 카테고리 분류 결과 저장
- 회신 필요 여부 및 기한 정보 저장

### PDF 자동 분류 및 저장
- 분류 결과 기반으로 카테고리 폴더 자동 지정
- PDF 파일을 해당 폴더로 자동 이동
- 업무협조 / 보고서 / 회의록 / 공지 별 PDF 관리

### 메일 현황 대시보드 (Streamlit)
- CSV 파일 업로드 후 대시보드에서 현황 확인
- 분류별 메일 수 차트 제공
- 검색 / 정렬 / 필터링으로 원하는 조건의 메일 조회

## ⚠️ 한계점 및 개선 방향

### 한계점
- 키워드 기반 분류 방식이라 문맥 반영에 한계가 있음
- 메일 수가 적어 분류 정확도 검증이 제한적임
- PDF 텍스트 품질에 따라 추출 결과가 달라질 수 있음

### 개선 방향
- 정답 레이블을 추가하여 실제 분류 정확도 측정
- 메일 데이터를 확대하여 다양한 유형의 메일 검증
- 머신러닝 기반 분류 모델 적용 검토
